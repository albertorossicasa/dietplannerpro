Istruzioni per compilare e ottenere APK / IPA
Passo 1: Prepara il progetto
Crea una cartella sul desktop, ad esempio DietPlannerPro-Standalone.

All'interno, crea le sottocartelle assets, src, src/database, src/contexts, src/screens, src/components, src/utils.

Copia tutti i file sopra nelle rispettive posizioni.

Apri un terminale in quella cartella ed esegui:

bash
npm install
Passo 2: Build per Android (APK)
bash
npx expo login   # se non già loggato
eas build -p android --profile preview
Alla fine, scarica l'APK dal link fornito.

Passo 3: Build per iOS (IPA)
Su Mac (obbligatorio):

bash
eas build -p ios --profile preview
Scegli Simulator (gratuito) se vuoi testare solo su simulatore, oppure Archive per distribuzione su App Store (richiede account Apple Developer a pagamento).
